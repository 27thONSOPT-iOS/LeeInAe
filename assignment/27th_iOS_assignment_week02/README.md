# 2주차 과제
## Frame & Bound
+ Frame

    상위 뷰(superView)의 좌표 시스템 안에서 view의 위치와 크기를 나타내는 좌표 시스템

+ Bounds

    자신만의 좌표 시스템으로 좌표를 나타냄
    `스크롤 == bounds가 변한다`

+ CGPoint

    2차원 좌표계 (x, y)

+ Content Offset

    bounds의 (x, y) 좌표를 의미 

## Delegate ?
대리자의 개념. swift에서 버튼, 텍스트 필드, 라벨 등의 객체가 존재하고, 각 객체는 고유의 특징을 가지고 있다. 이 델리게이트는 처리해야 할 일 중의 일부를 다른 객체에 넘기는 것을 뜻한다.
